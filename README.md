# Debian Desktop XFCE with Atom as IDE for Python (Vagrant)

## Installation

1. [Download & Install Vagrant](https://www.vagrantup.com)
2. `vagrant plugin install vagrant-vbguest`
3. `vagrant up`

* User: `debian`
* Password: `debian`
